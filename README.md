# Customer-Behavior-Analysis-SQL-Python-Power-BI

## End-to-End Customer Behavior Analytics Project

An end-to-end data analytics project focused on understanding customer purchasing patterns, shopping behavior, and business insights using **Python, SQL, and Power BI**.

This project demonstrates the complete analytics workflow—from data cleaning and exploratory analysis to database querying and interactive dashboard development.

---

## Project Overview

This project analyzes customer shopping behavior across multiple dimensions including:

* Customer demographics
* Purchase patterns
* Product preferences
* Subscription behavior
* Revenue contribution
* Discount utilization
* Customer segments
* Shopping trends

The objective is to transform raw customer transaction data into meaningful insights that can support strategic business decisions.

* **18 Features**

* Purchase Information
* Product Categories
* Shipping Preferences
* Subscription Status
* Review Ratings

---

## Key Insights

✔ Clothing emerged as the highest revenue-generating category.

✔ Young Adults contributed the largest share of revenue.

✔ Loyal customers represented the majority customer segment.

✔ Discount-dependent categories indicate promotional sensitivity.

✔ Subscription users showed spending patterns comparable to non-subscribers.

✔ Express shipping customers exhibited slightly higher average spending behavior.
